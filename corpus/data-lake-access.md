# Data Lake Access Runbook

To request data lake access, follow the runbook at internal/runbooks/data-lake-access.md.
Submit an access request via the Identity portal, request the DL-Reader entitlement, and
have your manager approve. Access to the curated zone is usually granted within one
business day once the manager approves.

## Steps

1. Open the Identity portal and start a new access request.
2. Request the **DL-Reader** entitlement for the curated data-lake zone.
3. Provide the business justification and the datasets you need.
4. Your manager approves the request; the data platform team provisions a scoped role.

Raw-zone access is a separate, higher-tier request — see the production data runbook.
