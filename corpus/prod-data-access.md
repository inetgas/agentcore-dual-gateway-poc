# Production Data Access Runbook

Production data access requires the same baseline as data-lake access plus a security
review (ProdData runbook §3). The approval chain includes Data Governance, not just your
manager. Expect two to three business days because of the added review.

## Procedure (ProdData runbook §3)

1. Complete the standard data-lake access request (DL-Reader entitlement).
2. Open a production-data addendum and attach the security-review questionnaire.
3. Data Governance reviews the request alongside your manager's approval.
4. On approval, a time-boxed production entitlement is granted and audited.

Production access is reviewed quarterly and auto-expires if unused for 60 days.
